# ゴママヨ検出器 使用方法

## 概要
このアプリケーションは、日本語テキスト中の「ゴママヨ」を検出するウェブアプリです。

「ゴママヨ」とは、前の単語の読みの最後の文字と、次の単語の読みの最初の文字が一致する現象のことです。

## 使い方

1. `index.html` をウェブブラウザで開きます
2. 辞書の読み込みが完了するまで待ちます（数秒かかります）
3. テキストエリアに日本語のテキストを入力します
4. 「検出する」ボタンをクリックします
5. 検出されたゴママヨがリスト表示されます

## 動作例

### 入力例1: "ゴマを使ったマヨネーズ"
検出結果:
- ゴマ + マヨネーズ (読み: ゴマ + マヨネーズ, 一致: マ)

### 入力例2: "東京は快適な都市です"
検出結果の例:
- 快適 + な (読み: カイテキ + ナ, 一致: キ/ナ) ※実際の検出結果は文章と形態素解析により異なります
- お好みの日本語テキストで試してみてください

## 技術仕様

- **形態素解析**: kuromoji.js v0.1.2
- **辞書**: https://takuyaa.github.io/kuromoji.js/demo/kuromoji/dict/
- **CDN**: jsdelivr
- **動作環境**: モダンブラウザ（Chrome, Firefox, Safari, Edge等）

## 注意事項

- インターネット接続が必要です（kuromoji.jsと辞書ファイルをCDNから読み込むため）
- 初回起動時は辞書の読み込みに時間がかかります
- ブラウザのコンソールでエラーが出る場合は、CDNがブロックされていないか確認してください

## セキュリティに関する注意

- 本アプリケーションは外部CDNからJavaScriptライブラリを読み込みます
- 本番環境で使用する場合は、Subresource Integrity (SRI)ハッシュの追加を推奨します
- 辞書ファイルも外部から読み込まれるため、信頼できる環境での使用を推奨します
- ユーザー入力はHTMLエスケープ処理されており、XSS攻撃に対する保護が施されています
